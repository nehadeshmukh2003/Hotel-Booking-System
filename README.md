# 🏨 Luxury Stay Hotel Booking System

A full-stack hotel booking web application built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
- Chart.js

---

## 🚀 Features

✅ Room Booking System  
✅ Admin Login  
✅ Approve / Delete Booking  
✅ Booking Status (Pending / Approved)  
✅ Dashboard Analytics  
✅ Chart.js Visualization  
✅ Search Filter  
✅ Mobile Number Validation  
✅ Popup Booking Form  
✅ Responsive Design  

---
